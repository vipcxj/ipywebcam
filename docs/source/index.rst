
ipywebcam
=====================================

Version: |release|

A Custom Jupyter Widget Library for Web Camera using WebRTC


Quickstart
----------

To get started with ipywebcam, install with pip::

    pip install ipywebcam

or with conda::

    conda install ipywebcam
